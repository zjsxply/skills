# 仓库规则

- 本仓库中的 skill 必须保持与具体 Agent 无关。
- 不要在共享文档、示例、元数据或提示词中提及任何具体的 Agent 品牌、产品名称或面向特定厂商的安装参数。
- 统一使用中性表述，例如 `agent`、`兼容 skill 的代理` 或 `客户端`。
- 如果某个命令支持可选的 Agent 专属参数，除非是在仓库外部编写私有的定向说明，否则不要把这些参数写入本仓库文档。
- 只有以下仓库级文件需要双语维护：`AGENTS.md`、`AGENTS.zh.md`、`README.md`、`README.zh.md`。
- Skill 内容默认只保留英文，除非未来仓库规则明确要求其他语言版本。
- 更新仓库级共享文档时，需保持中英文版本内容对齐。
- 英文版本见 [AGENTS.md](AGENTS.md)。
